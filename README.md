# SpeedManager

**Version:** 1.0.0
**Author:** the-xorcist
**Released:** 2026-02-15

## Description

Customizable movement speed adjustments

## Features

- **Global Speed**: Set a base movement speed for all entities
- **Player Speed**: Override speed specifically for your character
- **NPC Speed**: Override speed for NPCs and monsters
- **SimPlayer Speed**: Override speed for SimPlayers (AI companions)
- **Grouped SimPlayer Speed**: Special speed for SimPlayers in your party

## Based On / Inspiration

This mod is based on [Erenshor Walking Speed Adjust Mod by Lenzork](https://thunderstore.io/c/erenshor/p/Lenzork/Erenshor_Walking_Speed_Adjust_Mod/) and the Game Server Global move speed settings.  

Enhancements include:
- Separate speed controls for different entity types
- Special handling for grouped SimPlayers
- Cleaner Harmony patch implementation

## Installation

1. Install [BepInEx](https://docs.bepinex.dev/articles/user_guide/installation/index.html) for Erenshor
2. Copy the `.dll` file to your `BepInEx/plugins` folder
3. Copy the `MovementSpeedManager.cfg` file to your `BepInEx/config` folder
4. Launch the game
